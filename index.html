Futoast

A lightweight, customizable toast notification library for modern web apps

Get Started View Examples

Why Choose Futoast?

Futoast is a lightweight, feature-rich toast notification library designed for modern web applications with framework support.

Lightweight

Just 5KB gzipped with zero dependencies. Perfect for performance-critical applications.

Framework Support

Works with React, Vue, Angular, Django, and plain JavaScript out of the box.

Customizable

Easily customize appearance, behavior, and animations to match your app's design.

Responsive

Adapts to different screen sizes with mobile-specific positioning and styling.

Rich Features

Progress bars, drag & drop, buttons, sounds, dark mode, and much more.

Simple API

Intuitive API with shortcut methods for common notification types.

Installation

Futoast can be installed via CDN, npm, or downloaded directly.

CDN
NPM
Download

Using CDN

<script src="https://cdn.jsdelivr.net/npm/futoast@1.0.0/dist/futoast.min.js"></script>

Using NPM

npm install futoast

Import in your project

import Futoast from 'futoast';

Download

# Download from GitHub
wget https://github.com/yourusername/futoast/releases/download/v1.0.0/futoast.min.js

Then include it in your HTML:

<script src="path/to/futoast.min.js"></script>

Basic Usage

After including Futoast, you can start showing notifications with a simple API call.

Basic Example

// Show a simple notification
Futoast.show('Hello, World!');

// Show a success notification
Futoast.success('Operation completed successfully!');

// Show an error notification with options
Futoast.error('Something went wrong!', {
    timeout: 8000,
    position: 'top-center'
});

Futoast provides dedicated integration for popular JavaScript frameworks.

React Hook

import { useFutoast, FutoastContainer } from 'futoast';

function MyComponent() {
    const futoast = useFutoast();
    
    const handleClick = () => {
        futoast.success('Action completed!');
    };
    
    return (
        <div>
            <button onClick={handleClick}>Show Toast</button>
            <FutoastContainer />
        </div>
    );
}

Vue Plugin

import Vue from 'vue';
import Futoast from 'futoast';

// Install plugin
Vue.use(Futoast.Vue);

// In component
export default {
    methods: {
        showToast() {
            this.$futoast.success('Action completed!');
            // or this.$toast.success('Action completed!');
        }
    }
}

Angular Service

import { Component } from '@angular/core';
import { Futoast } from 'futoast';

@Component({
    selector: 'app-my-component',
    template: `<button (click)="showToast()">Show Toast</button>`
})
export class MyComponent {
    constructor(private futoast: Futoast) {}
    
    showToast() {
        this.futoast.success('Action completed!');
    }
}

Django Template

{% load static %}

<!-- Include Futoast -->
<script src="{% static 'js/Futoast' %}"></script>

<!-- Show toast from Django messages -->
{% if messages %}
    {% for message in messages %}
        <script>
            Futoast.{{ message.tags }}('{{ message }}');
        </script>
    {% endfor %}
{% endif %}

Positioning

Notifications can be positioned in different areas of the screen.

Position Options

// Top right (default)
Futoast.show('Top right', { position: 'top-right' });

// Top center
Futoast.show('Top center', { position: 'top-center' });

// Top left
Futoast.show('Top left', { position: 'top-left' });

// Bottom right
Futoast.show('Bottom right', { position: 'bottom-right' });

// Bottom center
Futoast.show('Bottom center', { position: 'bottom-center' });

// Bottom left
Futoast.show('Bottom left', { position: 'bottom-left' });
Mobile Behavior: On mobile devices, you can specify a different position using the mobilePosition option (either 'top', 'bottom', or 'same').

Custom Icons

Theming

Customize the look and feel to match your application's design.

Colors
Dark Mode
Fonts

Custom Colors

// Change primary color
Futoast.applyTheme({
    primary: '#8a2be2' // Purple
});

// Change multiple colors
Futoast.applyTheme({
    primary: '#8a2be2',
    success: '#28a745',
    error: '#dc3545',
    warning: '#fd7e14',
    info: '#17a2b8'
});

// Change background and text
Futoast.applyTheme({
    bg: '#f8f9fa',
    text: '#212529',
    border: '#dee2e6'
});

Dark Mode

// Set dark mode
Futoast.setThemeMode('dark');

// Set light mode
Futoast.setThemeMode('light');

// Toggle based on system preference
if (window.matchMedia('(prefers-color-scheme: dark)').matches) {
    Futoast.setThemeMode('dark');
} else {
    Futoast.setThemeMode('light');
}

Custom Fonts

// Set custom font family
Futoast.setFont('"Comic Sans MS", cursive');

// Set custom font size
Futoast.setFont('Arial', '14px');

// Reset to default
Futoast.setFont('', '');

Responsive Behavior

Futoast automatically adapts to different screen sizes with mobile-specific options.

Responsive Options

// Different position on mobile
Futoast.show('Top on desktop, bottom on mobile', {
    position: 'top-right',
    mobilePosition: 'bottom'
});

// Custom width on mobile
Futoast.show('Custom mobile width', {
    maxWidth: '350px',
    mobileMaxWidth: '95%',
    mobileMinWidth: '90%'
});

// Disable responsiveness
Futoast.show('Same on all devices', {
    mobilePosition: 'same',
    mobileMaxWidth: '350px'
});
Tip: Resize your browser window to see how the toasts adapt to different screen sizes.

Action Buttons

Futoast.show('Action required', {

            text: 'Retry',

API Reference

Complete reference of all available methods and options.

Methods

Method Description Returns
show(message, options) Display a toast notification Promise
primary(message, options) Display a primary toast (shortcut) Promise
secondary(message, options) Display a secondary toast (shortcut) Promise
success(message, options) Display a success toast (shortcut) Promise
error(message, options) Display an error toast (shortcut) Promise
warning(message, options) Display a warning toast (shortcut) Promise
info(message, options) Display an info toast (shortcut) Promise
question(message, options) Display a question toast (shortcut) Promise
loading(message, options) Display a loading toast (no timeout) Promise
clear() Remove all active toasts void
setIcons(icons) Change default icons Futoast
applyTheme(themeConfig) Change color theme Futoast
setThemeMode(mode) Set light/dark mode ('light' or 'dark') Futoast
setFont(fontFamily, fontSize) Change font family and size Futoast

Options

Option Type Default Description
position string 'top-right' Position of toast ('top-right', 'top-left', 'top-center', 'bottom-right', 'bottom-left', 'bottom-center')
timeout number 5000 Time in ms before toast disappears (0 = no timeout)
type string 'primary' Toast type ('primary', 'secondary', 'success', 'error', 'warning', 'info', 'question', 'loading')
theme string 'light' Color theme ('light' or 'dark')
icon string|boolean null Custom icon (string/HTML) or false to hide
closeButton boolean true Show close button
pauseOnHover boolean true Pause timeout on hover
draggable boolean false Allow dragging the toast
progressBar boolean true Show progress bar
sound boolean false Play default sound for type
html boolean false Allow HTML in message
animationIn string 'fadeIn' Animation when showing ('fadeIn', 'slideInRight', 'slideInLeft', 'zoomIn', 'bounce', 'flip')
animationOut string 'fadeOut' Animation when hiding ('fadeOut', 'slideOutRight', 'slideOutLeft', 'zoomOut')
onClick function null Callback when toast is clicked
onClose function null Callback when toast is closed
onShow function null Callback when toast is shown
onTimeout function null Callback when toast times out
rtl boolean false Right-to-left layout
css string '' Custom CSS styles for toast
queue boolean true Queue multiple toasts or show simultaneously
zIndex number 999999 z-index for toast container
overlay boolean false Show overlay behind toasts
overlayClose boolean false Close all toasts when overlay is clicked
overlayColor string 'rgba(0,0,0,0.5)' Color for overlay
customIcon string null Custom icon (overrides icon option)
customSound string null URL to custom sound file
buttons array null Array of button objects (text, action, class, style, closeOnClick)
focus boolean true Focus toast when shown
fontFamily string '' Custom font family for toast
fontSize string '' Custom font size for toast
maxWidth string '350px' Maximum width of toast
minWidth string '300px' Minimum width of toast
mobilePosition string 'top' Position on mobile ('top', 'bottom', 'same')
mobileMaxWidth string '90%' Maximum width on mobile
mobileMinWidth string '80%' Minimum width on mobile
ariaLive string 'assertive' ARIA live attribute ('assertive', 'polite', 'off')

Framework Examples

React Example

import React from 'react';
import { useFutoast, FutoastContainer } from 'futoast';

function LoginForm() {
    const futoast = useFutoast();
    
    const handleSubmit = async (e) => {
        e.preventDefault();
        
        try {
            futoast.loading('Logging in...');
            await login(username, password);
            futoast.success('Login successful!');
        } catch (error) {
            futoast.error(error.message);
        }
    };
    
    return (
        <>
            <form onSubmit={handleSubmit}>
                {/* form fields */}
                <button type="submit">Login</button>
            </form>
            <FutoastContainer />
        </>
    );
}

Vue Example

<template>
    <button @click="handleClick">Save Changes</button>
</template>

<script>
export default {
    methods: {
        async handleClick() {
            try {
                this.$futoast.loading('Saving changes...');
                await saveChanges();
                this.$futoast.success('Changes saved successfully!');
            } catch (error) {
                this.$futoast.error('Failed to save changes');
            }
        }
    }
}
</script>
